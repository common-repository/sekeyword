Display content depending on search engine and keywords

== Description ==

This Plugin lets you set some shortcodes and display content depending on what search engine the visitor came from and what the person searched for. 

This way you can make sure you can show your visitor content they are looking for.  

== Installation ==

1. Upload `index.php` to the `/wp-content/plugins/sekeyword` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
* or
1. Search for sekeyword via Plugins - Add New in the admin area
2. Click on Install
3. Activate plugin
